Projeto: EcoGuia Procopense

Um site que integra todas as atividades comunitárias de Cornélio Procópio e apresenta ao usuário
os demais projetos, como pontos de coletas, feiras e jardins, de forma simples e direta, mostrando
os dias de funcionamento e os locais onde estas atividades se encontram.

Este projeto também apresenta uma tela de login e outra de cadastro, para que um usuário possa
acessar seu perfil e adicionar suas atividades.

Ainda há muito a fazer para melhorar o projeto, mas para atingir o objetivo da atividade que é 
aplicar os conhecimentos adquiridos ao decorrer da aplicação da disciplina, este projeto cumpriu
com este propósito perfeitamente.
